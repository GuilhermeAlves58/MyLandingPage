## Personal Developer Card

Landing page pessoal desenvolvida com **HTML e CSS**, apresentando minhas principais tecnologias, links profissionais e currículo para download.

O projeto possui foco em design moderno, identidade visual e organização clara das informações.

---

### Tecnologias e Recursos Utilizados

- Gradiente animado em tons de azul  
- Animações suaves com `@keyframes`  
- Botões estilizados com efeito *hover* e microinterações  
- Layout responsivo para desktop e dispositivos móveis  
- Download direto do currículo em PDF  
- Estrutura simples e organizada em HTML semântico  

---

### Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Estruturação semântica em HTML  
- Estilização moderna com CSS puro  
- Animações e transições visuais  
- Responsividade utilizando `max-width` e `media queries`  
- Organização visual e hierarquia de informações

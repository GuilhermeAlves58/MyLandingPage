## Personal Developer Card

Landing page pessoal desenvolvida com **HTML e CSS**, apresentando minhas principais tecnologias, links profissionais e currículo para download.

### Tecnologias e Recursos Utilizados

- Gradiente animado em tons de azul  
- Animações suaves com `@keyframes`  
-  Botões estilizados com efeito *hover*  
-  Layout centralizado com foco em simplicidade e identidade visual  
- Download direto do currículo  
- Estrutura preparada para futura responsividade  

### Objetivo do Projeto

Este projeto foi desenvolvido com o objetivo de praticar:

- Estruturação semântica em HTML  
- Estilização moderna com CSS  
- Animações e transições  
- Organização visual e hierarquia de informações
